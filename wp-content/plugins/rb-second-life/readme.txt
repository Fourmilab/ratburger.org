This plug-in provides a widget, usually installed in a sidebar, which shows users present in a site's Second Life clubhouse.

== Description ==

This plug-in has no settings.  You can configure the widget title and population file it monitors when the widget is added to the sidebar.

1. Widget which displays users present in a Second Life clubhouse equipped with the Fourmilab Security Camera.
    
== Installation ==

1. Copy the entire /rb-second-life/ directory into your /wp-content/plugins/ directory.
2. Activate the plugin.
3. A "RB Second Life" widget will appear in Dashboard/Appearance/Widgets.
4. Drag the widget to the desired location in the sidebar or content section.
5. Specify your desired widget title and the file where the Security Camera logs active users.
