This plug-in shows all of a user's comments across all posts on a site, from the most recent to the first, on multiple pages.

== Description ==

Plug-in's settings can be changed at Settings > RB My Comments

1. This plug-in shows all of a user's comments on all posts.
2. Shortcode to display comments is [rb_my_comments]
3. Settings include comments per page, ascending/descending order by date, etc.
    
== Installation ==

1. Copy the entire /rb-my-comments/ directory into your /wp-content/plugins/ directory.
2. Activate the plugin.
3. A Dashboard item "RB My Comments" will appear.
4. Adjust the settings to those you prefer.
5. Use [rb_my_comments] shortcode where you want to display the comments.
